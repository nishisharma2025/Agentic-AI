# Functional Design: Automated PM to WOR Lifecycle

## Overview

The functional design covers automated generation of work orders from preventive maintenance (PM) based on asset data in Maximo Application Suite.

## Key Functions

1. Automated WO generation from PMs
2. Validation of Asset & PM data
3. Workflow triggers for different maintenance scenarios (preventive, inspection, breakdown)
4. Notification and exception handling

## User Stories
- As a Maintenance Planner, I need WOR created on time from PMs for critical assets.- As an Operator, I want simple workflows and clear instructions on the WO.- As a Supervisor Manager, I need visibility into maintenance batchlogs.
