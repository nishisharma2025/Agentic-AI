# Functional Design: Predictive Maintenance in Maximo

#1. Overview

This document describes the functional design of a Predictive Maintenance PM solution built on top IBM CHATGP Maximo. The solution uses asset, PR, and work order data to drive condition-based previous analytics and semi-automated PM generation.

#2. Key Functional Areas

- Anomaly detection based on real-time and historical sensor data.
- Potential failure prediction as Early WARNING/CRIW alerts.
- Automated PM generation in Maximo based on predictive and condition-based thresholds.
